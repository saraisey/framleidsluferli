The project

The project description, along with the description of the process of making it, can be found on this page. This was a group project.

My part in the project

Drawing

Ragnheiður and I tried to split the work load evenly for this project. As I could not download the VCarve program to my computer, she did all the drawing and toolpath making in this project, although we were on a zoom call helping each other out the entire time.

Production

We went to FabLab together to mill the project, and split the work that needed to be done there equally between us.

Documentation

As a part of the FabLab trip, I documented the process along with helping with the production itself. I wrote text that described the process of production and took photos of it aswell. Ragnheiður then coded the process afterwards.
