Making a website

Objective

In this first project, students were supposed to make a functional, well organized website in which the student introduced themselves and provided a resume.

My approach

Since this is the first time I have ever worked with HTML before, I started by watching videos the teacher had provided for the students. I then went on to choose a template from html5up.net, and settled on the one called Forty.

I next went on to download Brackets, which I used to modify the html files until I was happy with the result. Brackets is a very user-friendly open source text editor, which I found easy to understand. It offers a live preview of the website which I found extremely useful when modifying the html files.
