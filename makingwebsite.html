Making a website

Objective

In this first project, students were supposed to make a functional, well organized website in which the student introduced themselves and provided a resume.

My approach

Since this is the first time I have ever worked with HTML before, I started by watching videos the teacher had provided for the students. I then went on to choose a template from html5up.net, and settled on the one called Forty.

I next went on to download Brackets, which I used to modify the html files until I was happy with the result. Brackets is a very user-friendly open source text editor, which I found easy to understand. It offers a live preview of the website which I found extremely useful when modifying the html files.

Uploading website

I next went on to make an account on GitHub.

I then made a new repository and named it vefsida2021, making sure that it was public. Then I downloaded Git by installing Homebrew, and then installing Git through Homebrew (I use a macOS system). After having done that, I opened a terminal at the folder my files were stored in, and created a new repository from the command line as GitHub suggests when a new repository is made. I then finally changed the name of my repository so that the link to my website would be to my liking.

Updating website

Whenever I had made changes to the any of the html files, I used the commands git add ., git commit -m "message" and git push in the terminal to update the files in the repository.
