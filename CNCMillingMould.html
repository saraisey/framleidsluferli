CNC Milling | Making a mould

The project

The project description, along with the description of the process of making it, can be found on this page. This was a group project.

My part in the project

Drawing

Ragnheiður and I tried to split the work load evenly for this project. As Ragnheiður did all the drawing and toolpath making in project 4, I took it upon myself to draw and make toolpaths for project 5.

Production

There was no production part in this project, as time did not allow for it.

Documentation

I documented the process of the drawing part of the project while I drew it, along with taking videos of the screen to show the process and final results. Everything else was written together. Ragnheiður then coded the website for us. All in all it has been a very pleasant experience working with Ragnheiður and I found that we split the workload quite evenly.
